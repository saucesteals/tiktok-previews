# Tiktok Previews
## _Automatically send in-app videos of tiktoks sent in any chat!_

---

## 

## Installation  ⚙️

Tiktok Previews requires [Node.js](https://nodejs.org/) to run.

Install the dependencies.
```sh
yarn install
```

Create and configure an `.env` file from the following template:
```sh
# Your discord bot's token
DISCORD_BOT_TOKEN=
```

Start it!
```sh
# Start in development
yarn dev

# OR

# Watch in development
yarn watch

# OR 

# Start in production
yarn build
yarn start
```


## Usage

1. Make sure the bot has the following permissions in the channels you want it to work in:
      - View Channel
      - Send Messages
      - Embed Links
      - Attach Files
      - Read Message History
 
2. Send a link to any tiktok
